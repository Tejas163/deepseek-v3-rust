What is This?

A comprehensive implementation of DeepSeek V3.2 in Rust using the Candle ML framework. Learn the complete architecture from scratch!

Key Components Implemented

Multi-Head Latent Attention

KV cache compression into latent space for efficient long-context processing.

Mixture of Experts

256 experts with aux-loss-free load balancing. Only 8-9 activated per token.

DeepSeek Sparse Attention

Reduces O(L²) to O(L·k) complexity using lightning indexer.

Beam Search

Complete implementation from scratch - greedy, beam search, sampling.

Quick Start

# Clone and build
git clone https://github.com/Tejas163/deepseek-v3-rust
cd deepseek-v3-rust

# Run examples
cargo run --example train
cargo run --example inference
